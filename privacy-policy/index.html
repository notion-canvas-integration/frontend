C2N Privacy Policy

We know you care about your privacy. We care about it too. Let's make this quick:

  • We do not knowingly collect personal information.
  • Canvas tokens are not stored, and can easily be set to expire.
  • The Notion OAuth token is stored in the local storage on your device.
  • We will only use the Notion API to access the tools you've integrated it with, and only for the reasons that we clearly list on each tool's page.
  • We may collect analytics data from this website (e.g from Google Analytics), but not ever from the contents of your Notion pages - those are not read, only written to.
  • If you want to verify this or run your own version, our source code is public here.
